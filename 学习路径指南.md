## Stage 0:跑起来(Day 1)

```bash
# 1. 配置环境
cp .env.example .env
# 编辑 .env,填入 DEEPSEEK_API_KEY 和 ANTHROPIC_API_KEY

# 2. 装依赖
cd backend
python -m venv venv && source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt

# 3. smoke test(不调用真实 API)
pytest tests/

# 4. 跑一次完整流程
python -m co_scientist.cli run --question "如何让 RAG 减少 LLM 幻觉"
```

如果跑通,说明所有连接都 OK。开始按下面顺序读源码。

## ❓ 常见问题

**Q: pip install 卡在哪里?**
A: `psycopg2-binary` 在 Windows 偶尔有问题,可以先注释掉(走 SQLite 不用 Postgres)。

**Q: arXiv 报 429?**
A: 改 `arxiv_src.py` 里的 `delay_seconds` 增大间隔。

**Q: Claude 调不通?**
A: 确认 ANTHROPIC_API_KEY 和模型名。如果只想跑通,可以先把模块 4 的 META_REVIEWER.model_role 改成 "reasoner",纯 DeepSeek 也能跑(降级模式)。

**Q: 前端 ws 连不上?**
A: 确认后端跑在 8000 端口,且没有被防火墙拦。
