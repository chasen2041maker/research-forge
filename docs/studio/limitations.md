# Research Studio limitations

- Studio suggestions, citations, critiques, generated code, and dry runs are not Forge-verified
  research results.
- Its in-memory run snapshots are a UI/API convenience, not Forge business truth.
- Studio does not create a Forge Mission without a human-completed frozen specification.
- A future verified-result report must read Forge's completed source of truth rather than duplicate
  mutable Mission status inside Studio.
