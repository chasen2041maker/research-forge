# Report 文档入口

这组材料原本把“技术方案、架构设想、学习笔记、面试稿”分别展开，内容重叠较多。现在以 **[AI Co-Scientist 项目总览（精简版）](./AI-Co-Scientist_项目总览_精简版.md)** 作为唯一默认入口；先读它即可了解项目，而不必通读全部原稿。

## 建议阅读顺序

1. [项目总览（精简版）](./AI-Co-Scientist_项目总览_精简版.md)：10 分钟了解定位、架构、模块和依赖。
2. `../backend/co_scientist/graph.py`：以代码确认主工作流。
3. `../backend/co_scientist/api/main.py`：了解 API、进度上报与运行状态。
4. 有明确目的时，再查阅 `原始材料/` 中的专题稿。

## 范围说明

本目录记录的是旧版 **AI Co-Scientist**：一个用于学习和展示的 LangGraph 科研工作流。仓库当前的生产主线是 `backend/research_forge/`（Research Forge），两者没有运行时依赖关系；不要把本目录中的远期设想当作 Research Forge 已实现功能。

原稿均移动到 `原始材料/` 保留，未删除任何内容。
